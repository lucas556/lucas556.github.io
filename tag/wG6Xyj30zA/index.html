#NVCC

Linux安装CUDA NVCC

nvcc 其实就是 CUDA 的编译器，可以从 CUDA Toolkit 的 /bin 目录中获取，类似于 gcc 就是 c 语言的编译器。由于程序是要经过编译器编程成可执行的二进制文件，而 cuda 程序有两种代码，一种是运行在 cpu 上的 host 代码，一种是运行在 gpu 上的 device 代码，所以 nvcc 编译器要保证两部分代码能够编译成二进制文件在不同的机器上执行.
